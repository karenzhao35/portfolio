# My Portfolio

Personal portfolio website showcasing my projects and experience! All assets and vectors were designed by me in [Figma](https://www.figma.com/proto/2aXgXd6BPDDW7waomNvWSA/portfolio?node-id=410-368&t=x13SrZi5QYHvbLEV-1).

## Tech Stack

- **React** + **TypeScript**
- **Vite** - Build tool
- **Tailwind CSS v4** - Styling
- **Framer Motion** - Animations

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Structure

```
src/
├── components/   # Reusable components
├── pages/        # Page sections (Hero, About, Projects, etc.)
└── assets/       # Images and SVGs
```
